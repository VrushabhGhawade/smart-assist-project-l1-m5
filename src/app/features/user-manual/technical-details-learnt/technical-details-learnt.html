Technical Skills Learned – L1M4

  • Implemented Angular Lifecycle Hooks:
    • Loaded user and ticket data in ngOnInit
    • Accessed Material Dialogs in ngAfterViewInit
    • Cleaned subscriptions using ngOnDestroy
    • Proper subscription management to avoid memory leaks
  • Built Service-Based Architecture using Dependency Injection:
    • UserService
    • TicketService
    • MenuService
    • FilterStateService
    • Moved business logic from components to services
    • Components only consume Observables (thin components)
  • Used RxJS Observables for Reactive Programming:
    • Users → Observable<User>
    • Tickets → Observable<Ticket[]>
    • Filters → Observable<FilterState>
    • Async pipe for automatic subscription handling
  • Applied RxJS Operators:
    • map – transform and filter ticket data
    • filter – filter tickets by status
    • scan – compute ticket counts
    • switchMap – reload data on role/filter change
  • Followed Reactive & Clean Architecture Principles:
    • No nested subscriptions
    • No business logic inside components
    • Used BehaviorSubject for shared state
    • Centralized data management in services
    • Fully reactive UI updates
  • Integrated Angular Material Components:
    • MatTable with reactive data
    • MatDialog for feedback submission
    • MatChips for filtering
    • MatPaginator with reactive pagination
